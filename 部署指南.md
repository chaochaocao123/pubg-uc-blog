## 📦 网站文件

网站完整代码包已上传到项目目录：`/pubg-uc-blog/pubg-uc-blog.tar.gz`

解压命令：
```bash
tar -xzvf pubg-uc-blog.tar.gz
```

---

## 🚀 部署步骤（Netlify 演示）

### 第一步：下载并解压
1. 下载 `pubg-uc-blog.tar.gz`
2. 解压到本地

### 第二步：注册 Netlify
1. 访问 https://netlify.com
2. 使用 GitHub/邮箱注册账号
3. 登录后点击 "Add new site"

### 第三步：部署网站
1. 选择 "Deploy manually"
2. 拖拽解压后的 `pubg-uc-blog` 文件夹
3. 等待上传完成（约10秒）
4. 获得一个免费域名，如 `random-name-123.netlify.app`

### 第四步：更新域名配置
用你获得的实际域名替换以下文件中的 `pubg-uc-tips.netlify.app`：

1. `index.html` - 搜索 "pubg-uc-tips.netlify.app" 替换
2. `sitemap.xml` - 替换所有URL
3. 每篇文章HTML - 替换 canonical 标签

### 第五步：重新部署
修改完成后，再次拖拽上传文件夹即可。

---

## ⚡ 推荐：使用自己域名

在 Netlify 的 Site settings → Domain management 中添加你自己的域名（如 pubguc.com），可以：
- 获得免费SSL证书
- 自定义域名更专业
- 有利于SEO
